## Run Locally

Clone the project

```bash
  git clone https://github.com/MehakB7/react-table-assignment.git
```

Go to the project directory

```bash
  cd react-table-assignment
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```

## Features

- Global filtering
- column filtering
- sorting on price
